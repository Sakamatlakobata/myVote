Introduction

This project contains the following components:
  • Blog - Posts about meetings and bills
  • Polls - Voting on bills
  • Events - Arrangements for meetups
  • District - Returns your voting district and details about your representative
  • Campaigns - Assist campaigns in recording contact with voters
  • Messaging - Managing messaging such as SMS and Email
 The simple User Interface is a mobile-first design

Please create an account (even with bogus info) and log on, since most of the functionality is not available if not logged on (Members => Register New User)

Blog

Blog posts can be created for bills and/or events, and can be classified under categories (which can be maintained)

We can add new blog posts with the button, or maintain existing ones by clicking on the post title

We can add / delete comments on each blog post, and like the post in the detail (drilldown) view

Columns can be sorted by clicking on the title

Polls

Here we can vote on bills introduced to congress

We can cast votes on each bill, and see the vote count of everyone

To add more bill, please use the button. Likewise to delete bills from the list, as long as no votes have been cast

Events

These are tools to arrange meetings in congressional districts and ridings

The event list shows date and time, as well as venue, and the meeting organiser. Venues can be added as required, and assigned to events

By drilling down on an event (click on the event title), we can add ourselves to the list of attendees. The Attending button lists the events we are attending, and the headings sort columns. Also note that past events are removed

Members

The Members dropdown on the menu provides for registration, logging in, and maintenance of our accounts

The Voting District finds our congressional district from the zipcode, and displays links to monitoring our representative

One can still view blogs, polls, and events even if not logged in, but cannot maintain them. Registration is under the Members dropdown on the menu


Usage

This software is free to use

Please forward ideas on how we can make this more useful for you in your efforts to organise participation in our democracy

Please note that this is not a production instance, so there might be some data loss


Privacy

The developers of this project do not use your account data for any purpose

However this project is hosted in the cloud, so we cannot vouch for the cloud provider, or their staff

Please note that the email address and 'phone number entered on registration is visible to the planet, so perhaps it's best to start with dummy values (they can be changed at any time)
